  3. Derive the Doppler effect formula for sound $f' = f \left( \frac{v + v_o}{v - v_s} \right)$.

  4. Derive the intensity of a sound wave $I = \frac{p_0^2}{2 \rho v}$.

  5. Derive the inverse square law for intensity $I = \frac{P}{4 \pi r^2}$.
